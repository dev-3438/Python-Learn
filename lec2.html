üîÅ Loops

Essentially, loops are a way to do something over and over again.
Begin by typing code cat.py in the terminal window.
In the text editor, begin with the following code:

print("meow")
print("meow")
print("meow")
meow
meow
meow

Running this code by typing python cat.py, you‚Äôll notice that the program meows three times.
Loops enable you to create a block of code that executes over and over again.

‚è≥ While Loops

The while loop is nearly universal throughout all coding languages. Such a loop will repeat a block of code over and over again.

i = 0
while i < 3:
print("meow")
i += 1
meow
meow
meow

Notice how changing the operator to i < 3 allows our code to function as intended. We begin by counting with 0 and it iterates through our loop three times, producing three meows.

üîÅ For Loops

A for loop is a different type of loop. To best understand a for loop, it‚Äôs best to begin by talking about a new variable type called a list in Python.

print("meow")
meow
meow
meow

Notice how clean this code is compared to your previous while loop code.

üìè Length

We can utilize len as a way of checking the length of the list called students.

üìñ Dictionaries

dicts or dictionaries are a data structure that allows you to associate keys with values.
